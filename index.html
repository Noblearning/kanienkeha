Spelling system
Kanehsatà:ke — orthography used in this course. Uses í (not y), ę/ǫ for nasal vowels, and accent marks for stress and falling tone.
A note on dialect: This course is built on the Kanehsatà:ke dialect, home of some of the oldest living speakers of Kanien'kéha. Other communities use equally valid forms (e.g., Akwesasne uses l where Kanehsatà:ke uses r; Six Nations and Tyendinaga use y where Kanehsatà:ke uses i). No dialect is more "correct" than another. Learn more about dialect variation ↗
🔊 Audio note: The play buttons below use your device's text-to-speech engine to approximate each sound. No browser currently has a Kanien'kéha voice — these clips use carefully chosen phonetic approximations as a starting guide only (e.g., the long vowel a: is approximated as "aaah"). Always prioritise audio from community speakers at kanehsatakevoices.com ↗ and FirstVoices ↗.
